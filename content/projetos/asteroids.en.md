I made this game in 5 days to get some practice, after spending some time away from game development. I wanted to make a very simple game that I could finish quickly, so I decided to create a clone of the classic game [Asteroids](https://en.wikipedia.org/wiki/Asteroids_(video_game)) and add a few upgrades.

Music made by [Matthew Pablo](https://opengameart.org/content/space-boss-battle-theme), inspired by [Zigorous](https://www.youtube.com/@Zigurous)'s tutorial

The game is available for free on [itch.io](https://gamiru.itch.io/asteroids-and-steroids).
