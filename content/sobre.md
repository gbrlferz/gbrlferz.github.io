{{% details title="O que _gabiru_ significa?" closed="true" %}}

É um apelido derivado do meu nome real, Gabriel.

Além disso, é uma palavra com diversos significados.

- Gabiru é quem age com esperteza, o malandro.
- Gabiru é o alegre, o brincalhão, o gaiato.
- Gabiru é um dos nomes do demônio.
- Gabiru é quem vive do jogo (hehe).

Onde eu nasci, no nordeste do Brasil, é como chamam o rato-preto.

{{% /details %}}
